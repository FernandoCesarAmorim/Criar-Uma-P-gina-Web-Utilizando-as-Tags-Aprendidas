<h1>

(Volar)

Essas tags são usadas para definir títulos e subtítulos em HTML;

<h1>é o título de maior importância e <h6> é o de menor importância. Por exemplo, é o de menor importância. Por exemplo, <h1> Este é um título</h1> produzirá um título grande e em negrito.

<p>

(Volar)

Esta tag é usada para definir um parágrafo. Por exemplo, <p> Este é um parágrafo.</p> produzirá um bloco de texto que é tratado como um único parágrafo.

<mark>

(Volar)

Esta tag é usada para destacar ou marcar um texto. Por exemplo, <p> Este é um <mark>textodestacado</mark>.</p> fará com que “texto destacado” tenha um fundo amarelo.

<Small>

(Volar)

Esta tag é usada para definir um texto menor. Por exemplo, <p>Este é um <small>texto pequeno</small>.</p> ará com que “texto em itálico” seja exibido em itálico.

<i>

(Volar)

Esta tag é usada para definir um texto em itálico. Por exemplo, <p>Este é um <i>texto em itálico</i>.</p> fará com que “texto em itálico” seja exibido em itálico.

<u>

(Volar)

Esta tag é usada para definir um texto sublinhado. Por exemplo, <p>Este é um <u>texto sublinhado</u>.</p> fará com que “texto sublinhado” seja exibido com um sublinhado.

<strong>

(Volar)

Esta tag é usada para definir um texto em negrito. Por exemplo, <p>Este é um <strong>texto em negrito</strong>.</p> fará com que “texto em negrito” seja exibido em negrito.

<ol>

(Volar)

Esta tag é usada para criar uma lista ordenada. Por exemplo, <ol><li>Item 1</li><li>Item 2</li></ol>

  1. Item 1
  2. Item 2
produzirá uma lista com números.

<ul>

(Volar)

Esta tag é usada para criar uma lista não ordenada. Por exemplo, <ul><li>Item 1</li><li>Item 2</li></ul>

  • Item 1
  • Item 2
    • produzirá uma lista com marcadores.

    <li>

    (Volar)

    Esta tag é usada para definir um item da lista. Deve ser usada dentro de uma tag <ol> ou <ul>.

    <a>

    (Volar)

    Esta tag é usada para criar um link. Por exemplo, <a >Este é um link</a> produzirá um link clicável que leva a “https://www.example.com”.

    <hr>

    (Volar)

    Esta tag é usada para criar uma linha horizontal, que pode ser usada para separar seções de conteúdos

    <sub>

    (Volar)

    Esta tag é usada para definir um texto subscrito. Por exemplo, <p>H<sub>2</sub>O</p> produzirá “H2O”.

    <sup>

    (Volar)

    Esta tag é usada para definir um texto sobrescrito. Por exemplo, <p>E = mc<sup>2</sup></p> produzirá “E = mc2”.

    <blockquote>

    (Volar)

    Esta tag é usada para definir uma seção que é citada de outra fonte. Por exemplo,

    <blockquote>Citação famosa</blockquote>
    um bloco de texto que é tratado como uma citação

    <font>

    (Volar)

    Embora seja obsoleta em HTML5, essa tag era usada para definir a fonte, tamanho e cor do texto. Por exemplo,<font face="verdana" color="green" size="5">Este é um texto verde em Verdana.</font> produziria um texto verde na fonte Verdana de tamanho 5.

    <del>

    (Volar)

    Esta tag é usada para mostrar um texto que foi deletado ou substituído. Por exemplo, Este é um <del>texto deletado</del>. produzirá “texto deletado” com um traço através dele.

    <abbr>

    (Volar)

    Esta tag é usada para definir uma abreviação ou acrônimo. Quando você passa o mouse sobre o texto marcado com essa tag, o título completo é exibido. Por exemplo, <abbr>title="Cascading style Sheets>CSS</abbr> produzirá “CSS” e quando você passa o mouse sobre ele, “Cascading Style Sheets” será exibido.
